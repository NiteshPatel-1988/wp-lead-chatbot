A simple lead-generation chatbot for WordPress that collects user details and stores them in your admin dashboard.

== Description ==
NP Lead Chatbot is a simple plugin that lets you:

== Features ==
* Capture leads directly from a chatbot interface.
* Store chat and lead data in custom tables and display it in the admin panel.
* Export leads as CSV
* Secure and lightweight


Ideal for businesses that want a straightforward lead collection system via a chatbot.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/np-lead-chatbot` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to NP Lead Chatbot in the admin dashboard to view and manage collected leads.

== Frequently Asked Questions ==
= How are leads stored? =
Leads are stored in custom database tables and can be viewed in the admin dashboard.

= Can I customize the chatbot? =
Basic customization is possible via the plugin settings in the admin dashboard.
